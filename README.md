Student Management System

Description

A console-based Student Management System developed in C++ using Object-Oriented Programming and File Handling.

Features

- Add Student Records
- Display Student Records
- Search Student by ID
- Update Student Information
- Delete Student Records
- Persistent Storage using Binary Files

Technologies Used

- C++
- OOP Concepts
- File Handling

How to Run

Compile:
g++ main.cpp -o sms

Run:
./sms
